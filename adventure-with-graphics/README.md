# Adventure with Graphics

**Adventure with Graphics** is a graphics-based programming project developed as part of **Stanford CS106AX**. The project combines **algorithmic reasoning** with **graphical rendering and interaction**, emphasizing correctness, structure, and creative problem-solving.

This project was awarded **first place in a course-wide graphics competition**.

---

## Overview

The goal of the project was to design an interactive graphical program that integrates:

* Structured program flow
* State management
* Visual feedback and user interaction

The emphasis is on **clean abstraction and reliable behavior**, rather than visual complexity alone.

---

## Key Features

* Interactive graphical elements rendered programmatically
* Event-driven logic responding to user input
* Clear separation between game state, rendering logic, and control flow
* Robust handling of state transitions and redraws

---

## Concepts Demonstrated

* Graphics programming and event handling
* Algorithmic control of visual systems
* Modular program design and abstraction
* Debugging graphical and state-based programs

---

## How to Run

```bash
Adventure.py
```

(Requires the CS106AX graphics support library and Python 3.)
